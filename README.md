📝 Daily Routine (React + TypeScript)

Projeto de lista de tarefas desenvolvido para praticar React com TypeScript,
focando em componentização, tipagem e persistência de dados no navegador.

🚀 Funcionalidades:
- Componentização da interface para melhor organização e manutenção
- Tipagem de props e estados com TypeScript
- Persistência de tarefas usando localStorage
- Reset automático das tarefas concluídas ao detectar um novo dia
- Interface responsiva e estilização com Tailwind CSS
- Ícones com Lucide React

🛠️ Tecnologias:
- React
- TypeScript
- Tailwind CSS
- Lucide React

📚 Aprendizados:
- Comunicação entre componentes via props
- Uso de useEffect para lógica baseada em tempo
- Modelagem de estado com TypeScript
- Organização de um projeto React de médio porte
